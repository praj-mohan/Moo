The following files are in this program

echoTest: executable of the program
cow.wav: the original audio
echo.wav: the "echoed" version of the audio


function files:
array: copies the values of cow.wav into an array 
copyfile: copies the values of the array into echo.wav
main: calls functions and initializes the array
modifyfile: modifies the array to give the "echo" effect
reader: reads the cow.wav file to find the total size of the file.
